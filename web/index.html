Quick Start

Configuration is stored in .ovcs/ovcs.json in the watched directory.

Server mode:

ovcs-server                   # start the server (default port 5984)
ovcs-server --port 4000       # custom port

Client mode (server sync):

ovcs                          # watch current directory
ovcs --dir /path/to/project   # watch a specific directory
ovcs --port 3002              # custom web dashboard port

Client mode (P2P):

ovcs --p2p                    # peer-to-peer via WebRTC
ovcs --p2p --dir /path        # P2P + custom directory

Minimal config for server sync:

{
  "email": "you@example.com",
  "remote": "http://localhost:5984/ovcs",
  "teamId": "my-team",
  "sync": { "enabled": true }
}

Minimal config for P2P:

{
  "email": "you@example.com",
  "remote": "http://localhost:5984/ovcs",
  "teamId": "my-team",
  "p2p": { "signalingServer": "ws://localhost:5984/signaling" }
}
